Loan Approval Prediction System

A machine learning-powered web application for automating loan eligibility checks.

Python Flask Machine Learning MySQL

1. Project Overview

The Loan Approval Prediction System streamlines the loan application process by using historical data to predict whether an applicant is eligible. It reduces manual verification time and provides instant feedback to users.

The system includes a public-facing portal for applicants and a secure dashboard for bank administrators.

2. Key Features

For Applicants (Users)

  • Instant Eligibility Check: AI-driven prediction (Approved/Rejected) based on income and credit score.
  • Application Tracking: Check status using a unique Application ID.
  • AI Chatbot: A built-in assistant to answer queries 24/7.
  • Responsive UI: Works seamlessly on mobile and desktop.

For Administrators

  • Secure Login: Password-protected admin dashboard.
  • Data Visualization: View all applicants in a structured table.
  • Report Generation: Print capabilities for offline records.

3. Technology Stack

Component Technology
Frontend HTML5, CSS3, JavaScript (Chatbot)
Backend Python (Flask Framework)
Database MySQL (XAMPP/Localhost)
ML Model Scikit-Learn (Random Forest/Logistic Regression)

4. System Architecture

The system follows the MVC (Model-View-Controller) pattern:

  1. View: HTML forms collect user data.
  2. Controller: Flask (Python) processes the input.
  3. Model: The ML Model (`model.pkl`) predicts the outcome.
  4. Database: Results are stored in MySQL for the Admin Dashboard.

5. Database Schema

Database Name: loan_database

Table: applicants

Column Type Description
id INT (PK) Auto-incrementing ID
full_name VARCHAR Applicant Name
applicant_income DECIMAL Monthly Income
loan_status VARCHAR Prediction (Approved/Rejected)
application_date TIMESTAMP Submission Date

Table: admins

Column Type Description
username VARCHAR Login Username
password VARCHAR Login Password

7. Setup Guide

Follow these steps to run the project locally:

Step 1: Install Dependencies

pip install flask flask-mysqldb pandas numpy scikit-learn

Step 2: Setup Database

Open MySQL Workbench and run the provided SQL script to create loan_database.

Step 3: Run Application

python app.py

Visit http://127.0.0.1:5000/ in your browser.

8. User Manual

Applying for a Loan

  1. Click Apply Now on the homepage.
  2. Fill in your income, loan amount, and credit history.
  3. Submit to see if you are Approved or Rejected instantly.

Admin Access

  1. Click Admin Login (Lock Icon).
  2. Login with admin / admin123.
  3. View, manage, and print the list of applicants.
